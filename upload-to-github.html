Upload These Files to GitHub

Pick Web UI or Git CLI. 5 minutes.

Home

Option A — GitHub Web UI (no terminal)

  1. Create a repo at github.com/new → name it education-hub → set to Public or Private → Create repository.
  2. On the repo page, click Add file → Upload files.
  3. Drag the contents of the education_hub/ folder into the drop area (or upload the folder if your OS allows).
  4. Add a commit message like Initial hub → click Commit changes.
  5. Optional: enable GitHub Pages (Settings → Pages → Source: Deploy from a branch → Branch: main / root). Your site will publish at https://USER.github.io/education-hub/.

Option B — Git (Terminal)

unzip education_hub_MAX.zip
cd education_hub
git init
git add .
git commit -m "Initial Education Hub"
git branch -M main
git remote add origin https://github.com/USER/education-hub.git
git push -u origin main
