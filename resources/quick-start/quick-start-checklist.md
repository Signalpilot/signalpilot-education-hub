## Troubleshooting Checklist

### If Indicators Don't Load

- [ ] Check TradingView subscription (Pro+ needed for many features)
- [ ] Verify invite-only access granted
- [ ] Refresh the page
- [ ] Try different browser
- [ ] Clear browser cache

### If No Signals Appear

- [ ] Increase timeframe (try 4H or Daily)
- [ ] Check volatile asset (BTC, SPY)
- [ ] Zoom out to see more history
- [ ] Verify settings not filtering signals

### If Confused About Signals

- [ ] Re-read Pentarch Complete Reference
- [ ] Watch video tutorials
- [ ] Ask in Discord #education channel
- [ ] Review this checklist from Phase 5
