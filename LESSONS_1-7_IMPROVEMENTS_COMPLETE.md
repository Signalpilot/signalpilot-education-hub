## Key Patterns Discovered

### ✅ **ADD Precision** (Lessons 2-3)
Technical lessons need:
- Specific thresholds (+3,000, +5,000, 50%)
- Visualizations (tables, footprints, tape)
- Concrete examples with exact numbers

### ✅ **CUT Redundancy** (Lessons 4-6)
Weekly timelines that repeat the same pattern:
- Week 1: Lost money
- Week 2: Lost money
- Week 3: Lost money...
Summary stats already tell this story!

### ✅ **PRESERVE Educational Repetition** (Lesson 7)
Multi-angle storytelling is GOOD when:
- Same event shown from different perspectives
- Each iteration teaches something new
- Psychology/neuroscience lessons benefit from this

### ✅ **FIX Inconsistencies**
TL;DR must match actual case studies:
- Wrong names (Emily vs Jake, Tom vs Sarah)
- Wrong numbers ($15.6K vs $9.8K)

## What Makes Lessons Captivating (Learnings)

1. **Concrete numbers** > Vague concepts ("50% higher volume" not "volume spike")
2. **Visual proof** > Text descriptions (footprint tables, not "buy/sell imbalance")
3. **One detailed example** > Ten repetitive summaries (Feb 14 trade not 10 weeks)
4. **Tight storytelling** > Exhaustive documentation (cut the bloat)
5. **Multi-angle teaching** > Single-pass explanation (for psychology lessons)
