Trading Journal Setup & Weekly Review

Beginner18 minGeneral education

Educational content, not financial advice.

Learning Objectives
  • Create a practical journal with tags & screenshots.
  • Run weekly reviews to tighten rules.
  • Track process metrics.
Core Concepts
  • What to Log: Setup, bias, entries/exits, risk, screenshots, emotions.
  • Tagging: Consistent tags for setups and mistakes.
  • Metrics: Win rate by setup, avg R, errors, time‑of‑day.
  • Weekly Review: Top mistakes/strengths; change log.
Examples
  • Neutral screenshots speed learning.
  • Process metrics reveal leaks early.
Checklist
  • Log within 15 minutes of exit.
  • Change rules only via a written log.
Exercises
  1. Create tag taxonomy for 3 setups & 3 mistakes.
  2. Write your first weekly review.
