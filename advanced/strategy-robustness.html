Robustness: OOS, Walk‑Forward & Monte Carlo

Advanced26 minGeneral education

Educational content, not financial advice.

Learning Objectives
  • Validate with OOS & WFA.
  • Use Monte Carlo for path risk.
  • Test parameter stability.
Core Concepts
  • OOS: Holdout data the model never sees.
  • WFA: Roll optimize→test windows.
  • Monte Carlo: Randomize sequences to get drawdown distributions.
  • Sensitivity: ±20% parameter shifts shouldn’t kill it.
Examples
  • If OOS collapses, edge likely overfit.
Checklist
  • Prefer simple models; set risk overlay by worst‑case DD.
Exercises
  1. Run 1000‑path Monte Carlo on Rs.
  2. Vary core parameter ±20%.
Key Terms
TermDefinition
Parameter StabilityResilience to small tweaks.

Q1. Monte Carlo estimates:

  • A. Exact P&L
  • B. Outcome distributions
  • C. Broker fees
  • D. Taxes
