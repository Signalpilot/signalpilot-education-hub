Multi‑Timeframe Analysis & Alignment

Intermediate18 minGeneral education

Educational content, not financial advice.

Learning Objectives
  • Create a bias TF and an execution TF.
  • Resolve conflicts systematically.
  • Avoid over‑switching.
Core Concepts
  • Bias vs Execution: E.g., 1H bias, 5–15m entries.
  • Conflict Rules: Trade only with bias or stand aside.
  • Drift Windows: Allow time for alignment; avoid forcing.
Examples
  • Alignment reduces chop trades.
Checklist
  • Write explicit conflict rules.
  • Keep TF count to 2–3 max.
Exercises
  1. Build an alignment matrix for two TFs.
  2. Back‑review 20 trades for alignment.
Key Terms
TermDefinition
AlignmentBias and entries pointing same way.

Quick Quiz (self‑check)

Q1. Primary TF is used for:

  • A. Entries
  • B. Bias context
  • C. Broker routing
  • D. Screen size

Answer: B
