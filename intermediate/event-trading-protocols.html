Event Trading Protocols (Earnings, CPI, FOMC)

Intermediate22 minGeneral education

Educational content, not financial advice.

Learning Objectives
  • Create pre/post‑event rules.
  • Manage gaps and spreads.
  • Log outcomes specifically.
Core Concepts
  • Pre‑Event: Reduce risk/size; avoid late new trades.
  • During: Expect slippage; prefer limits or stand aside.
  • Post‑Event: Wait for stabilization; trade reclaims/continuations.
Examples
  • Bid/ask can widen dramatically around events.
Checklist
  • Use 'no‑trade windows' before key events.
  • Size down post‑event until spreads normalize.
Exercises
  1. Write an event protocol template.
  2. Collect 10 event cases.
Key Terms
TermDefinition
Gap RiskPrice jumps between bars without prints.

Quick Quiz (self‑check)

Q1. Common event behavior:

  • A. Tighter spreads
  • B. Wider spreads & volatility
  • C. Guaranteed up move
  • D. No effect

Answer: B
