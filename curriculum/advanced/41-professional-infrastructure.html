Data Quality Checks

import pandas as pd

def validate_data(df):
    # Check for gaps (missing bars)
    time_diff = df['timestamp'].diff()
    gaps = time_diff[time_diff > pd.Timedelta('2 minutes')]
    if len(gaps) > 0:
        print(f"Warning: {len(gaps)} gaps found")
    
    # Check for outliers (bad ticks)
    price_change = df['close'].pct_change()
    outliers = price_change[abs(price_change) > 0.05]  # 5%+ moves
    if len(outliers) > 0:
        print(f"Warning: {len(outliers)} outliers found")
    
    # Check for duplicates
    dupes = df.duplicated(subset=['timestamp'])
    if dupes.sum() > 0:
        print(f"Warning: {dupes.sum()} duplicate timestamps")

Risk Management System

Automated Risk Checks

Pre-Trade Risk System:

class RiskManager:
    def __init__(self, max_position_risk=0.02, max_portfolio_heat=0.08):
        self.max_position_risk = max_position_risk
        self.max_portfolio_heat = max_portfolio_heat
    
    def check_position_size(self, equity, entry, stop, size):
        risk_amt = (entry - stop) × size
        risk_pct = risk_amt / equity
        
        if risk_pct > self.max_position_risk:
            return False, f"Position risk {risk_pct:.1%} > max {self.max_position_risk:.1%}"
        return True, "OK"
    
    def check_portfolio_heat(self, equity, open_positions):
        total_risk = sum([pos.risk_amt for pos in open_positions])
        heat_pct = total_risk / equity
        
        if heat_pct > self.max_portfolio_heat:
            return False, f"Portfolio heat {heat_pct:.1%} > max {self.max_portfolio_heat:.1%}"
        return True, "OK"

Kill Switch

Automated Trading Halt Conditions:

1. Daily loss > 3% → Close all, stop trading
2. Drawdown > 20% → Reduce size 75%
3. Consecutive losses > 5 → Stop, review
4. API errors > 10/hour → Stop, investigate
5. VIX > 40 → Close all, sit out

Implementation:
- Check conditions every 5 minutes
- Send SMS/email alert when triggered
- Requires manual override to resume

Disaster Recovery Plan

Failure Scenarios

FailureBackup Plan

Emergency Protocol

If Primary System Fails:

1. Access positions via mobile app
2. Close all positions (or place stops)
3. Switch to backup system (laptop + hotspot)
4. Contact broker if execution issues
5. Log incident, review after market close

Drill: Test failover quarterly (simulate failures)
